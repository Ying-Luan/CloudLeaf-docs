## Project Overview

### Tech Stack

| Category | Technology |
| ------ | ------ |
| Framework | [Plasmo](https://www.plasmo.com/) |
| UI | React + Tailwind CSS v4 |
| Language | TypeScript |
| State Management | Zustand + Immer |
| Persistence | @plasmohq/storage |

### Local Setup

```bash
git clone https://github.com/Ying-Luan/CloudLeaf.git
cd CloudLeaf
npm install

# Start dev server
npm run dev          # Chrome
npm run dev:edge     # Microsoft Edge
npm run dev:firefox  # Firefox

# Build for production
npm run build          # Chrome
npm run build:edge     # Microsoft Edge
npm run build:firefox  # Firefox

# Package for store
npm run package          # Chrome
npm run package:edge     # Microsoft Edge
npm run package:firefox  # Firefox
```

### Architecture

```text
Popup / Side Panel / Settings (UI)
  └── hooks/                   State & side-effect encapsulation
      └── core/sync/           Sync orchestration (cloud / local)
          └── providers/       Storage backends (Gist / WebDAV / Local)
              └── BaseProvider  → HttpProvider  → Concrete implementations
```

## Contributing

We welcome all forms of contribution! Whether to the [code repo](https://github.com/Ying-Luan/CloudLeaf) or [docs repo](https://github.com/Ying-Luan/CloudLeaf-docs) — filing issues, submitting pull requests, or joining discussions — every contribution is appreciated.

### Filing Issues

Submit at [GitHub Issues](https://github.com/Ying-Luan/CloudLeaf/issues) with the following information:

- Browser and version
- Steps to reproduce
- Expected vs. actual behavior

### Pull Requests

1. Fork the repository
2. Create a new branch
3. Ensure `npm run lint` passes
4. Submit a PR with a description of your changes
