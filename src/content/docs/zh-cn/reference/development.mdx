## 项目概览

### 技术栈

| 类别 | 技术 |
| ------ | ------ |
| 框架 | [Plasmo](https://www.plasmo.com/) |
| UI | React + Tailwind CSS v4 |
| 语言 | TypeScript |
| 状态管理 | Zustand + Immer |
| 持久化 | @plasmohq/storage |

### 本地搭建

```bash
git clone https://github.com/Ying-Luan/CloudLeaf.git
cd CloudLeaf
npm install

# 运行开发服务器
npm run dev          # Chrome
npm run dev:edge     # Microsoft Edge
npm run dev:firefox  # Firefox

# 构建生产版本
npm run build          # Chrome
npm run build:edge     # Microsoft Edge
npm run build:firefox  # Firefox

# 打包发布文件
npm run package          # Chrome
npm run package:edge     # Microsoft Edge
npm run package:firefox  # Firefox
```

### 架构

```text
弹窗 / 侧边栏 / 设置页（UI）
  └── hooks/                   状态与副作用封装
      └── core/sync/           同步编排（cloud / local）
          └── providers/       存储后端（Gist / WebDAV / Local）
              └── BaseProvider  → HttpProvider  → 具体实现
```

## 贡献指南

我们欢迎任何形式的贡献！无论是向[代码仓库](https://github.com/Ying-Luan/CloudLeaf)还是[文档仓库](https://github.com/Ying-Luan/CloudLeaf-docs)提交 Issue、Pull Request，或是参与讨论，都非常感谢。

### 提 Issue

在 [GitHub Issues](https://github.com/Ying-Luan/CloudLeaf/issues) 提交，请包含：

- 浏览器及版本
- 操作步骤
- 预期与实际表现

### 提 Pull Request

1. Fork 仓库
2. 创建新分支
3. 确保 npm run lint 通过
4. 提交 PR，描述改动内容
